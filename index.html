My projects

MaskedImageModelling

PyTorch implementation of a self-supervised learning framework to pre-train a model for an image segmentation task. A VisionTransformer is pre-trained by learning to reconstruct partially masked images, and fine-tuned on the Oxford-IIIT Pets dataset for segmentation.

DeepRL

Implementation of various model-free deep reinforcement learning algorithms built with PyTorch. The project utilises OpenAI's gymnasium environments to train the algorithms, as well as producing rendered outputs to visualise the agent's performance.

PlaygroundML

Various common machine learning models and algorithms applied to toy datasets in Python, often with the aim of producing a visual representation of how they work. Easy to follow Jupyter Notebooks and nice gifs.
